# Projeto Biblioteca 📚

Projeto iniciante em Java que simula o gerenciamento de uma biblioteca pelo terminal.

## Funcionalidades

- Adicionar livros
- Remover livros
- Listar livros cadastrados
- Escolher o tipo/gênero do livro
- Organizar os livros dentro de uma biblioteca

## Tecnologias usadas

- Java
- Programação Orientada a Objetos
- Classes e objetos
- Métodos
- Construtores
- Arrays
- Enumeração
- Estruturas condicionais
- Laços de repetição
- Scanner para entrada de dados no terminal

## Objetivo do projeto

O objetivo desse projeto foi praticar conceitos básicos e intermediários de Java, criando um sistema simples de CRUD para uma biblioteca.

CRUD significa:

- Create: adicionar livro
- Read: listar livros
- Update: alterar dados, se for implementado depois
- Delete: remover livro
